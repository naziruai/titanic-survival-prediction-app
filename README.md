## 🌐 Live Demo
**Try the app here:** https://titanic-survival-prediction-app-naz.streamlit.app/

## 📱 Features
- Interactive web app works on **desktop and mobile**
- Real-time survival predictions
- 77% accuracy with Logistic Regression
- Visual probability indicators

## 🛠️ Technologies Used
- **Frontend:** Streamlit
- **Backend:** Python
- **ML Model:** Scikit-learn (Logistic Regression)
- **Deployment:** Streamlit Cloud

## 📊 Model Performance
- Accuracy: 77.1%
- Precision: 68.4%
- Recall: 75.4%
- F1-Score: 71.8%
