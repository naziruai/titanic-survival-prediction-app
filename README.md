# Titanic Survival Prediction App 🚢

## Overview
This project predicts whether a passenger would survive the Titanic disaster using Logistic Regression.

## Live Demo
[Click here to try the app](https://your-app-link.streamlit.app)

## Features
- Interactive web app built with Streamlit
- Real-time predictions based on passenger information
- 77% accuracy with optimized threshold (0.624)

## Technologies Used
- Python
- Streamlit
- Scikit-learn (Logistic Regression)
- Pandas, NumPy
- Matplotlib, Seaborn
